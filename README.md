## Core idea

The platform is designed to reduce price pressure in service markets by introducing confidential pricing and context-dependent offers, while simultaneously helping professionals prioritize their work based on time, urgency, and relevance.

# Rent a Pro

Rent a Pro is a prototype of a platform for self-employed professionals and their clients.

The idea behind the project is to enable direct booking of services without creating public price pressure. Prices are not openly listed for comparison across providers. Instead, customers select a service, define time and context, and receive an individual offer depending on the specific situation.

At the same time, the platform is designed to support self-employed professionals in managing their time more effectively. Requests are evaluated based on factors such as estimated time effort, urgency, and customer relevance. From this, the system generates a clear prioritization of incoming jobs. The goal is not only to simplify booking, but also to reduce organizational overhead and give professionals more time for their actual work.

## Current status

This repository contains an early functional prototype with:

- basic user registration and login
- service creation and listing
- simple booking flow
- priority score calculation for services
- backend API built with Express
- frontend built with React and Vite
- lightweight local JSON-based data storage

## Planned features

The broader concept includes further features such as:

- confidential and context-dependent pricing
- flexible pricing models
- map-based discovery of services and providers
- calendar synchronization
- real-time notifications
- verification for professional accounts
- mobile-first and app-ready frontend
